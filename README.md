🏎️ Ferrari Enthusiast Portal
A high-performance React application built to showcase the elegance, speed, and engineering mastery of Ferrari.
🌟 Overview
This project is a digital showroom designed to provide an immersive experience. I built this to practice component-based architecture and modern UI/UX principles, ensuring the website feels as fast and responsive as a Ferrari on a track.
🚀 Features

    Dynamic Gallery: Interactive display of iconic Ferrari models (SF90, LaFerrari, F8).
    Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.
    Performance Focused: Built using React for fast state transitions and smooth rendering.
    Rosso Corsa UI: A premium dark-themed interface with the signature Ferrari Red accents.

🛠️ Tech Stack

    Frontend: React.js
    Styling: CSS3 / SCSS (Custom Animations)
    Build Tool: Vite / Create React App
    Deployment: Dockerized and served via Nginx (CI/CD via Jenkins)
